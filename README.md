# Intro To Networking

### Ping
```bash
ping <target>
```

### TraceRoute 
It's to see the Path between you and the destination you set

```bash
traceroute Dest
```

### Whois & dig
This is a command to know about the Ip you want to
```bash
whois IP
dig Ip
```
